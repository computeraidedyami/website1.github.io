React

JavaScript library for building user interfaces (UI)

Fundamentals

Components

Isolated pieces of code to build user interfaces ,We use components to tell React what we want to see on the screen. When data changes, React will update an re-render our components.
The render method returns a description of what you want to see on screen.In particular returns a React Element, which is a lightweight description of what to render.
